In this step:

  Note

This step is only for workloads based on psets or fss groups. If you chose npar or vpar as your compartment type in the first step, you will skip this step, as gWLM can determine the workload automatically.

  1. Select existing workloads or create new ones by choosing one of the following options:

    • Select one or more of the existing workloads

      The existing workloads are not currently in an SRD. They may have been part of an SRD but were removed from the SRD through the Unmanage Workloads wizard.

    • Create new workloads

      When managing workloads based on psets or fss groups, create new workloads by specifying a number in the field labeled:

      Number of additional workloads to create

  2. Select the [Next] button.
