Overview   Step 1    Next step

The Manage New Systems wizard helps you quickly begin using gWLM when you:

  • Introduce gWLM into your environment

  • Add a new system to a gWLM environment

  Note

Once you have managed a new system using this wizard, do not return to the wizard to make changes for that system. See the gWLM home page, available by selecting the [gWLM Home] button in the top right corner of most gWLM windows, for pointers on making changes.

In this step:

  1. Enter the names of the systems where you will run workloads that you want gWLM to manage. Separate the names using spaces.

  2. Select the [Next] button.
