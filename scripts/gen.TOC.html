I have a perl script that takes an HTML file and produces a copy of that HTML file, but includes a TOC (table of contents) at the top of the copy.

Usage:
% gen.TOC.pl input.html > output.html

You maintain and update the input.html file.
Create the output.html file just for posting on the net.

The script is available on presskit at
/usr/local/doctools/bin/gen.TOC.pl

Jan 6, 2000

I have a variation of gen.TOC.pl:
/usr/local/doctools/bin/gen.TOC.numbered.pl

This script numbers the sections in the HTML file and includes those section numbers in the TOC.

Use this script the same as gen.TOC.pl. 
